# Simple GO server

This project contains the code to create a simple server using GO language.
The guide to create this server follows [this tutorial](https://blog.logrocket.com/creating-a-web-server-with-golang/) wrote by [Michiel Mulders](https://blog.logrocket.com/author/michielmulders/).

### Goals

This project:
- Web server that can accept GET requests;
- The web server will respond to a POST request;
- Basic securities at the server are aplied.


### Structure

The project is divided in "server.go" and the "static" folder.
- server.go: All the GO code is developed in this file.
- static folder: All the static code are included at this folder, where we have the HTML code.

### Running

To run this project, download this project and at the root, run the following line:
```
go run server.go
```
